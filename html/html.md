
HTML标签之a标签

- 超链接：较常用的功能

- 锚点：直接定位到id是demo1的位置

  ```javascript
  <a href="#demo1">find demo1</a>
  ```

- 打电话：美团、饿了么外卖等

  ```javascript
  <a href="tel:xxxxxxxxxxx">给xxx打电话</a>
  ```

- 发邮件

  ```javascript
  <a href="mailto:zhzhang09@126.com">给宝山发邮件</a>
  ```

- 协议限定符：直接写JavaScript代码

  ```javascript
  <a href="javascript:alert(1)">弹出1</a>
  ```
